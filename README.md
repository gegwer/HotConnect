# HotConnect

Рофл-игра про Гошу, преподавательницу колледжа, с выборами, мини-играми, спецэффектами и боями в духе Undertale.

## Как открыть

Открой `index.html` в браузере.

## Что есть

- короткая, но ветвистая история
- бои с кнопками `Удар`, `Действие`, `Предмет`
- мини-игра с листами и мини-игра с кофе
- босс-плесень из холодильника и финальный босс в стиле Undertale
- вспышки, тряска, смена тем и мрачный режим работы

## Управление

- `WASD` или стрелки - движение в бою
- `Space` или `Enter` - удар по таймингу и остановка в кофейной мини-игре
- `Esc` - сдаться в мини-игре с бумагой

## Что можно добавить дальше

- новые боссы
- секретные концовки
- ещё более тупые и смешные ветки сюжета
